# waermepumpe-rechner-v2

Premium rebuild of the Wärmepumpen Rechner landing page (Orion-inspired structure) with:
- hero + glass card
- trust strip
- structured benefit sections
- calculator centerpiece
- methodik + FAQ + CTA
- lead form posting to `/api/lead`

## Local
Open `index.html` directly for static preview.

## Deploy (Vercel)
```bash
vercel --prod
```

## Required env for API
- `SUPABASE_URL`
- `SUPABASE_SERVICE_ROLE_KEY`
